# 🤖 AI Resume Screening System

An intelligent web application that analyzes resumes against job descriptions and provides an **ATS-style match score**.
The system helps **recruiters quickly screen candidates** and helps **job seekers improve their resumes** by identifying missing skills.

This project is built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** and basic **NLP concepts for skill extraction and resume analysis**.

---

# 🚀 Features

### 👨‍💼 Job Seeker

* Upload resume (PDF/TXT)
* Analyze resume against job description
* View ATS match score
* Identify **matched skills and missing skills**
* Get recommendations to improve resume
* Download resume analysis report
* View analysis history

### 🧑‍💻 Recruiter

* Post job openings
* View candidate applications
* View ATS score of candidates
* Filter candidates by score
* Shortlist or reject candidates
* View analytics dashboard

### 🛠 Admin

* Manage users
* Manage recruiters
* View all resumes
* Monitor job postings
* View system analytics
* Access logs and system controls

# 📊 How It Works

1. User uploads a resume.
2. User enters a job description.
3. The system extracts skills from the resume.
4. Skills are compared with job description requirements.
5. The system calculates a **match score (ATS score)**.
6. The dashboard displays:

   * Match percentage
   * Matched skills
   * Missing skills
   * Improvement suggestions

# ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/vinayby06-cpu/AI-Resume-Screening-System.git
```

### 2️⃣ Navigate to Project

```bash
cd AI-Resume-Screening-System
```

### 3️⃣ Install Dependencies

Backend:

```bash
cd server
npm install
```

Frontend:

```bash
cd client
npm install
```


### 5️⃣ Run the Application

Start Backend:

```bash
node server.js
```

Start Frontend:

```bash
npm start
```
