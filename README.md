# 🤖 AI Resume Screening System

An intelligent web application that analyzes resumes against job descriptions and provides a skill-match score.

🚀 Built using MERN stack + NLP concepts for placement-focused learning.

---

## 🔥 Features

✅ Upload resume (PDF/TXT)  
✅ Enter job description  
✅ Automatic skill extraction  
✅ Match score calculation  
✅ Matched vs Missing skills  
✅ Clean animated UI  
✅ Production deployed  

---

## 🛠 Tech Stack

**Frontend**
- React.js
- CSS3
- Framer Motion
- React Circular Progressbar

**Backend**
- Node.js
- Express.js
- Multer (file upload)

**Deployment**
- Render (Frontend + Backend)
- GitHub

---

## 📊 How It Works

1. User uploads resume  
2. User enters job description  
3. Backend extracts predefined skills  
4. System compares skills  
5. Match score is generated  
6. UI displays analysis visually
